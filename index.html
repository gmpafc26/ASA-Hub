Request Process

How Retail, Commercial and PCL appearance requests are submitted and approved

🔔 How Requests Work
Submit your request via the + New Request tab. Select TBC if you're not yet certain, or Planned if you're ready to go. You'll receive a confirmation email when submitted, and a further email once your request is approved or not approved. Once internally approved, the relevant approver will notify the football department for final sign-off.
1
Submit your request via + New Request
Fill in the request name, date, location, start time, time required, players and any relevant notes.
🛍 Retail → Georgie or Andrew 💼 Commercial → Jodie or John 🌟 PCL → Jacob
2
Confirmation email sent automatically
You'll receive an email confirming your request has been submitted and is pending review.
3
Approver reviews the request
The relevant approver checks the calendar for availability and conflicts, then approves or declines.
4
Approval email sent to submitter
You'll be notified by email of the outcome. Status updates to Internally Approved or Not Approved.
5
Approver notifies the football department
For internally approved requests, the approver contacts the relevant football department liaison. AFL → Trent, AFLW → Rachael.
Click 📧 Football Dept on any Internally Approved request in the Requests tab.
6
Confirmed — fully locked in
Once the football department has confirmed, the approver updates the status to Confirmed.
📅
Request Status Guide
  • TBC — Submitted but not yet certain.
  • Planned — Certain and ready for the approver to review.
  • Internally Approved — Approved by the relevant approver. Football dept to be notified.
  • Not Approved — Declined. Check comments for the reason.
  • Confirmed — Fully locked in. Football dept confirmed.
Always check the calendar before submitting to avoid conflicts.

How It Works

Guide to using the ASA / CBA Player Requests Hub

🔒 Internal use only — do not share access or request information outside the organisation.
🤏
What is this hub?
A centralised portal for managing AFL and AFLW player appearance requests across Retail, Commercial and PCL streams. All requests, approvals, player availability and ASA/CBA balances are tracked in one place and synced live to the monday.com board.
🔐
Access levels
Anyone can view without logging in. Use the PIN button (top right) to access approver or admin features.
⚪ ViewerBrowse players, calendar and requests — no login required
🟢 CommercialJodie Cairns & John Kleeman — approve commercial requests
🔴 AdminGeorgie, Andrew, Simon & Jacob — full access including schedule, players & balances
👥
Players tab
Browse AFL and AFLW players with photo, jersey number, position, ASA type and remaining balance. Toggle between squads using the AFL / AFLW buttons. Click Requests on a player card to see their upcoming bookings. Admins can click Edit on any player card to update their details, photo URL, ASA type or restricted status. Use + Add Player to add a new player to the board.
📅
Calendar tab
View player availability and existing requests at a glance. Toggle between AFL, AFLW and Both views. Click any day to see full availability details and request bookings. Use + Blackout to add media blackout dates across a date range. Admins see an Edit Schedule button — clicking it opens the schedule manager. From a day's detail popup, admins can also click Edit next to any availability line to jump straight to editing that entry.
✉️
Making a request
Click + New Request in the top nav. Fill in the request name, type (Retail, Commercial or PCL), date, location, start time, duration, players and any notes. The date field shows AFL and AFLW availability for the selected day. Existing bookings for the same week are flagged on each player card. The request is automatically routed to the relevant approver on submission.
✅
Requests & approvals
The Requests tab shows all requests sorted by date, filterable by department, status and time period. Approvers see an amber banner when requests are pending review — click it to open the approval modal. Approved requests move to Internally Approved. Use the 📧 Football Dept button to email Trent or Rachael, then track progress using the football dept status dropdown (Not Sent → Sent → Football Dept Confirmed). Admins can edit any field on a request — including date, location, players and status — and all changes sync back to the board.
✍️
ASA / CBA balances
The Balances tab is visible to admins. It shows each player's remaining ASA/CBA percentage as a colour-coded bar (green = healthy, amber = getting low, red = low). Update a player's balance by typing in the number field — changed values turn amber. Click Save All to Board to sync all changes to monday at once. The last updated date is shown at the top and stamped on the board.
📅
Managing the schedule
Admins access the schedule via the Edit Schedule button in the Calendar tab (or the Edit link on any day detail popup). The schedule modal has two modes. Add Entry lets you add or edit a single date — pick a type (Not Available, Player Day Off, Player Travel Day, Available After/Before a time, Available All Day, CBA) and the note auto-fills. Typing a date that already has an entry pre-loads it for editing. Bulk Add loads a full month grid with a row per day — set each weekday individually, use Apply to all to fill all weekdays at once then tweak individually, and hit Save Month Schedule to sync everything to the board. Weekends are auto-filled. Existing entries show with an amber dot and pre-populate their current values. Toggle Overwrite existing to replace already-set entries.
📋
monday.com board sync
Every change made in this hub — new requests, edits, approvals, balance updates, player edits, schedule changes and media blackouts — is synced live to the ASA / CBA HUB board on monday.com. The board is always up to date and can be used as a source of truth. Note: editing your personal board notification settings for this board will not affect any automated email notifications, which run independently.
